05 — 统一生产流水线

7 步，全链路覆盖

三条轨道共用需求入口和质检交付出口，中间节点按类型自动路由。

1
结构化需求提交人工录入
  • Track 1：SKU / 文案 / 语言 / 目标尺寸
  • Track 2 战报：数据表格上传
  • Track 2 KV：活动名称 / 主题色 / 参考图
  • Track 3：创意简报 / 动图方向描述
2
AI 解析 + 路由Claude API

自然语言 → 结构化 JSON，判断类型，分配轨道。违禁词检测，置信度低升级半自动。

3
模板匹配规则 + 向量

规则引擎精确匹配，未命中走语义向量检索，相似度 >0.85 自动选用。战报类直接进渲染引擎。

4
分轨生产 T1 全自动 T2 半自动 T3 辅助
  • Track 1：JSON → PS UXP → 图层替换 → 多尺寸导出（全自动）
  • Track 2 战报：Figma Auto Layout 动态排版 → Make 触发 API 导出
  • Track 2 KV：Claude 生成配色 → 飞书 Bot 确认 → 脚本执行
  • Track 3 复杂 KV：Lovart 生成背景 → 设计师合成 → 转 Track 2 延展
  • Track 3 动图：即梦图生视频 → 导出 GIF/MP4
5
多尺寸批量导出全自动

每个尺寸有独立布局规则（非等比缩放），一次提交自动产出全部目标规格。

6
双层质检 规则 + Vision AI 5–20% 抽检
  • 规则：文字溢出 / 分辨率不足 / 违禁词 / 图层缺失
  • Vision AI：logo 遮挡 / 文字可读性 / 色彩对比度
  • Track 1 抽检 5%，Track 2 抽检 20%，Track 3 全量人工
7
交付归档全自动

物料库上传 + 命名归档 + 飞书通知业务方 + 生产数据入库，反向优化模板匹配。
